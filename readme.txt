# A2 Python Development Internship Assignment

This project scrapes Alibaba RFQ listings using Selenium in Python.

## What It Does:
- Extracts RFQ Titles and URLs
- Scrolls to load dynamic content
- Navigates across 10 pages using pagination

## Files Included:
- alibaba_rfq_scraper.py → Python script
- alibaba_rfq_output.csv → Output data (200 RFQs)
- README.txt → Instructions

## How to Run:
1. Install required packages:
   pip install selenium pandas webdriver-manager

2. Run the script:
   python alibaba_rfq_scraper.py
